# 🎧 Spotify Analysis

An exploratory data analysis project using Microsoft Excel to uncover insights from a Spotify dataset. The project is designed as part of my data analyst portfolio using Excel, SQL, and Power BI.

---

## 📌 Objectives

- Analyze Spotify listening patterns using Excel
- Practice data cleaning, transformation, and visualization techniques
- Summarize insights through clear and visual dashboards
- Document findings and share results as part of a data analytics portfolio

---

## 📂 Dataset

- **Source:** [Kaggle – Spotify Dataset](https://www.kaggle.com/datasets)
- **Format:** `.xlsx` / `.csv`
- **Contents:** Track metadata, popularity, genre, audio features (e.g., danceability, energy, tempo)

---

## 🛠️ Tools Used

- **Excel:** formulas, pivot tables, conditional formatting, Power Query
- *(Power BI and SQL may be integrated in future stages)*

---

## 🔍 Key Questions Explored

- Which tracks are the most popular across genres?
- How do audio features vary between genres or years?
- What correlations exist between energy, tempo, and popularity?

## 🐍 Python Analysis

A small utility script is provided in `scripts/hiphop_analysis.py` to explore Hip-Hop tracks using the built-in `csv` module. It computes average audio features and lists the most popular songs. Run it with:

```bash
python scripts/hiphop_analysis.py
```
